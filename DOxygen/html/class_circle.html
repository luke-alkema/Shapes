Detailed Description

The purpose of the Circle class is to extend from the Shape class and further model a Circle.

This class adds onto the Shape class by adding the data-member radius as well as the PI constant allowing an actual shape to be made from the inheritance relatioship between these 2 classes. Circle class adds multiple functions that make use of Pi and the circles radius to perform calculations which can be displayed to the user.
