Detailed Description

The purpose of the Square class is to extend from the Shape class and further model a Square.













This class adds onto the Shape class by adding the data-member sideLength. The square class adds multiple functions that make use of the side length and number of sides a square has to perform calculations such as area, perimeter, and the overalldimension, which in a squares case is the sidelength. There are also getters and setters for the sidelength data member.
