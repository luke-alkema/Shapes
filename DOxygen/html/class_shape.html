Detailed Description

The purpose of the Shape class is to model a shape to its most abstract level, its name and its colour. So specialized child classes to inherit from here.

The shape class holds the general properties of any shape. The purpose of this shape class is to allow child classes to inherit from it, of which are acual specific shapes.
The shape class can change the colour and name of the shape(red, square), and allow users to change and /or access the data members name and colour.
The class also has 3 pure virtual functions so that child classes can use the concept of them. They must be overridden of course since the methods will change given different shapes.
