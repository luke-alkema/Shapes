\doxysubsection{Detailed Description}
The purpose of the \doxylink{class_square}{Square} class is to extend from the \doxylink{class_shape}{Shape} class and further model a \doxylink{class_square}{Square}.

This class adds onto the \doxylink{class_shape}{Shape} class by adding the data-\/member side\+Length. The square class adds multiple functions that make use of the side length and number of sides a square has to perform calculations such as area, perimeter, and the overalldimension, which in a squares case is the sidelength. There are also getters and setters for the sidelength data member.
